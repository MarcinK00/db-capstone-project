# db-capstone-project  
Repository for the Database Engineer Capstone course project  

I wasn’t entirely sure about the best structure for this project, and I’m aware that the approach I chose is not ideal. However, I believe it still demonstrates that all tasks have been completed.  

## Project Structure  

### Modules 1 & 2  
The **db-capstone-project** file contains solutions for tasks from Modules 1 and 2.  

I created several `.md` files because the initial database I designed was not sufficient to successfully complete the subsequent tasks. For later tasks, I needed to make further changes to the database structure. In hindsight, it would have been better to update the original file instead.  

### Module 3  

- **Python file**: Contains solutions for Python-related tasks  
- **Tableau file**: Contains solutions for Tableau-related tasks  

## Notes  
This project evolved over time as requirements changed, which is why multiple files and database adjustments were necessary.
